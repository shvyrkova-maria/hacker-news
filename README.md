# hacker-news

###Dev
Запустить режим разработки

```shell
npm start
```

Во вкладке браузера перейти по адресу
[http://localhost:8081](http://localhost:8081)

###Building
Для того чтобы создать оптимизированные файлы для хостинга

```shell
npm run build
```

###Deploying/Publishing

```shell
npm run deploy
```
